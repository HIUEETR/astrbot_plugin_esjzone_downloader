## [1.3.0] - 2026-06-07

### 新增 ✨
- 速率限制器 (`RateLimiter`)
  - 基于令牌桶算法的速率控制
  - 防止服务器限流
  - 支持动态调整速率
- 进度追踪器 (`ProgressTracker`)
  - 实时下载进度追踪
  - 章节和图片统计
  - 下载速率计算
  - 定期进度消息推送
- 缓存管理器 (`CacheManager`)
  - 内存和文件双层缓存
  - LRU 淘汰策略
  - 自动过期清理
  - 并发安全访问
- 监控管理器增强 (`MonitorManager`)
  - 智能检查间隔（根据更新频率动态调整）
  - 失败重试指数退避
  - 监控历史记录保存
  - 优先级队列调度
  - 统计信息生成
- 错误处理器 (`ErrorHandler`)
  - 详细错误分类（网络、权限、资源、限制、解析）
  - 用户友好错误提示
  - 智能重试策略
  - 错误自动分类
- 性能监控模块 (`PerformanceMonitor`)
  - 下载速度统计
  - 章节和图片成功率统计
  - 实时性能报告生成

### 改进 🚀
- 下载性能优化
  - 异步下载池已完全集成
  - 章节下载并发控制（semaphore）
  - 图片下载独立队列
  - 下载进度聚合
- 监控系统增强
  - 智能检查间隔（24小时内更新缩短间隔，长期未更新延长间隔）
  - 批量下载优化（优先级队列）
  - 失败重试机制（指数退避，最多 8 倍基础间隔）
  - 监控历史记录（最多保留 1000 条）
- 用户体验优化
  - 下载进度消息（实时显示章节和图片进度）
  - 快捷命令已支持（`/esj d 123` 直接下载）
- 错误处理增强
  - 网络错误、权限错误、资源错误、限制错误自动分类
  - 自动重试（可配置次数和延迟）
  - 部分成功处理（跳过失败章节继续下载）
  - 用户友好提示（技术错误转换为中文提示）
- 代码结构优化
  - 模块职责明确（各模块独立职责）
  - 消除代码重复（提取公共逻辑）
  - 统一异步接口风格
- 类型注解和文档
  - 所有新增模块完整类型注解
  - 关键类和方法添加 docstring

### 文档 📚
- 新增 `ARCHITECTURE.md` - 完整架构文档
- 更新 `IMPROVEMENT_PLAN.md` - 标记所有已完成任务
- 新增模块文档
  - `rate_limiter.py` - 速率限制器
  - `progress_tracker.py` - 进度追踪器
  - `cache_manager.py` - 缓存管理器
  - `monitor_manager.py` - 监控管理器
  - `error_handler.py` - 错误处理器

## [1.2.0] - 2026-06-07

### 新增 ✨
- 异步下载管理器 (`AsyncDownloadManager`)
  - 基于 asyncio 的高效并发下载
  - 章节和图片分离队列
  - 智能任务调度和负载均衡
- 收藏缓存管理器 (`FavoritesManager`)
  - 本地 JSON 缓存机制
  - 会话内去重更新
  - 异步并发获取多页
- 工具函数库 (`utils.py`)
  - 文件名清理 `sanitize_filename`
  - 文本处理和格式化工具

### 改进 🚀
- 监控系统错误处理增强
  - 单点失败不影响其他监控
  - 更详细的错误日志
- 下载性能优化
  - 从线程池转换为异步协程
  - 降低内存占用
  - 更高的并发效率

### 修复 🐛
- 修复 Pillow 依赖版本冲突（移除 `<11` 限制，兼容 AstrBot 核心的 12.1.1）

### 文档 📚
- 新增 `MIGRATION_SUMMARY.md` - 迁移工作总结
- 新增 `README_v1.2.md` - 使用说明
- 更新 `IMPROVEMENT_PLAN.md` - 标记已完成任务

## [1.1.0] - 2026-06-07

### 新增 ✨
- 核心模块迁移到 AstrBot 插件架构
- 异步下载管理器基础实现
- 收藏管理器基础实现
- Cookie 多用户隔离管理

### 改进 🚀
- 依赖库统一和清理
  - 移除 CLI 工具依赖（questionary, rich, tqdm, colorama, wcwidth）
  - 移除配置文件依赖（pyyaml, ruamel.yaml）
  - 保留核心依赖（beautifulsoup4, httpx, pillow）

### 变更 ⚙️
- 日志系统适配为 AstrBot 的 logger API
- 配置系统适配为 AstrBot 的配置机制

## [1.0.0] - 2026-04-30

### 新增 ✨
- 基础下载功能（EPUB/TXT）
- 用户登录/登出管理
- 书籍信息查询（`/esj info`）
- 收藏列表获取（`/esj fav`）
- 更新状态检查（`/esj check`）
- 分章节下载（指定起始/结束章节）
- 配置管理（`/esj cfg`）
- 更新监控系统（`/esj m add/list/rm/check`）
- 多用户支持
- Cookie 管理
- 图片下载支持
- 文件命名模式配置
- 下载限制配置（章节数、图片大小等）

### 命令 🎮
- `/esj help` - 查看帮助
- `/esj info <URL>` - 查看书籍信息
- `/esj download <URL> [格式] [起始] [结束]` - 下载书籍
- `/esj login <邮箱> <密码>` - 登录账号
- `/esj logout` - 退出登录
- `/esj fav [排序] [页码]` - 查看收藏
- `/esj check <URL>` - 检查更新
- `/esj m add <URL>` - 添加监控
- `/esj m list` - 查看监控列表
- `/esj m rm <URL|all>` - 移除监控
- `/esj m check` - 立即检查更新
- `/esj cfg [配置项] [值]` - 配置管理

## 版本说明

### 版本号格式
- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能新增
- **修订号**: 向下兼容的问题修复

### 变更类型
- **新增**: 新功能
- **改进**: 现有功能的改进
- **修复**: Bug 修复
- **变更**: 不影响功能的变更
- **废弃**: 即将移除的功能
- **移除**: 已移除的功能
- **安全**: 安全相关的修复
